Promoting Library Accessibility and Outreach with Special Collections in Virtual Reality

A Lightning Talk for DH2020
By Julia Uhr


The Special Collections of CU Fairytale Library (SCCUFL) is a virtual reality library built using A-Frame and featuring illustrated and historical fairy tales from the University of Colorado Boulder Libraries' Department of Special Collections, Archives and Preservation. The aim of this project is to give students and the general public access to special collections books in an immersive and interactive virtual environment to increase engagement with the material.

SCCUFL can be viewed at juliauhr.github.io/sccufl.
The code is open-source and can be found at github.com/juliauhr/sccufl.
